# decodelabs-_tasks-2
## 🤖 Project 2: Supervised Learning (Classification & Regression)

### 📌 Project Overview
This project applies supervised machine learning algorithms to solve both classification and continuous prediction problems. The focus is on implementing robust predictive models through proper feature mapping, hyperparameter tuning, and strict mathematical evaluation metrics.

### 🛠️ Key Technical Requirements & Milestones
* **Classification Pipeline:** Built a classification model (such as Titanic Survival Prediction) using algorithms like Logistic Regression or Support Vector Machines (SVM). Implemented proper encoding for categorical features and handled class imbalances.
* **Regression Pipeline:** Developed a regression framework (such as Stock Price Prediction) using Linear Regression to forecast continuous values based on historical trends.
* **Model Optimization & Tuning:** Fine-tuned hyperparameters to prevent overfitting and underfitting, optimizing structural coefficients and decision boundaries.
* **Rigorous Evaluation:** Evaluated model performance using core statistical metrics including Accuracy, Precision, Recall, F1-Score, and Mean Squared Error (MSE).

### 🧮 Tech Stack & Skills
* Python (Pandas, NumPy)
* Scikit-Learn (LogisticRegression, SVM, LinearRegression)
* Model Evaluation (Confusion Matrix, F1-Score, MSE)
* Predictive Modeling & Hyperparameter Tuning
